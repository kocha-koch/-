Пути и методы

  • GET-запрос "GET /" - начальная страница, коренной путь
  • GET-запрос "GET /hello" - страница с приветствием
  • GET-запрос "GET /stats" - страница возращающая таблицу с полями user-agent и количеством запросов
  • GET-запрос "GET /comments" - возвращает сообщение клиента
  • POST-запрос "POST /comments" - возвращает сообщение клиенту
